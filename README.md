# S2 "Data processing and Storage" Project

## How to Deploy? 

1. Clone repo using Git
```
git clone https://github.com/Nikita-NN/SimpleBanking.git
```
```
cd SimpleBanking
```
2. Install Docker and Docker compose plugin
3. Use
```
docker compose up --build
```
